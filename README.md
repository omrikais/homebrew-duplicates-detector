# Homebrew Tap — duplicates-detector

Homebrew formulae and casks for [duplicates-detector](https://github.com/omrikais/duplicates-detector).

## Install

```bash
brew tap omrikais/duplicates-detector

# CLI tool
brew install duplicates-detector

# macOS GUI app (installs CLI automatically)
brew install --cask duplicates-detector-gui
```

## What's included

| Package | Type | Description |
|---------|------|-------------|
| `duplicates-detector` | Formula | CLI tool for detecting duplicate/similar video, image, or audio files |
| `duplicates-detector-gui` | Cask | Native macOS GUI companion app |
